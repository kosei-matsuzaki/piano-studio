### 音の再生（解説スライド）— 過不足なく出す方針

解説スライドは、内容に応じて音を**出す／出さない**を自動で切り替えます。

**① 概念デモ（鍵盤アニメ＋発音）** … 次のときだけ自動再生（`shouldAnimate`）:
- 音符・コードが**2つ以上**並ぶ（＝違い・流れ・音階・進行など動きがある）
- 単一でも**4音以上のリッチな和音**（7th／テンション。「◯◯の響き」系）
- それ以外（**単音・単一トライアド・音材なし**）は **楽譜のみ**を表示し、鍵盤は鳴らさない。
- `anim:true` / `anim:false` でスライド側から明示上書き可。

**② 応用例メロディ** … `example` を書いた**そのスライドだけ**「🎵 応用例を聴く」で再生（自動でも1回流れる）。
- スケール・旋法・カデンツ・進行など「**その概念を使うとどう聞こえるか**」が要点のスライドに、手書きで用意（きらきら星／ii-V-I／王道進行・カノン進行／ブルースのリック／各旋法の旋律 等）。
- すべてのスライドには付けない（過剰を避ける）。

### `example` の書き方（応用例メロディ）

```js
example: {
  tempo: 120,                         // BPM（省略時 100）
  chords: [ {offs:[0,4,7], beats:4},  // 伴奏。offs=主音からの半音オフセット、beats=長さ（拍）
            {offs:[7,11,14], beats:4} ],   // 省略可。[0,4,7] のように配列だけでも可（beats=4）
  melody: [ [0,1], [2,1], [4,2],      // [オフセット, 拍] の並び。主音相対なので選択キーに追従
            "r1", [7,2] ],            // 休符は "r<拍>" または {rest:true, beats}
  label: "きらきら星（Cメジャー）",
}
```

- `melody`/`chords` のオフセットは**主音からの半音数**（`practice` と同じ相対表現。負値・12超も可＝聴くだけなので広く取れる）。
- 伴奏は自動で1オクターブ下に鳴らす（低くなりすぎる場合は同オクターブ）。メロディは発音に合わせて鍵盤をハイライト。

### `rhythm` の書き方（リズム編）

```js
rhythm: {
  tempo: 100,           // BPM
  beats: 4,             // 1小節の拍数（拍子の分子。4/4=4, 3/4=3, 7/8なら8分7個=7）
  bars: 1,              // 小節数（2小節クラーベなら 2）
  grid: 4,              // 1拍の分割数（16分=4, 8分=2, 三連=3, 4:5ポリなら5）
  loop: 2,              // くり返し回数
  layers: [             // 打楽器レイヤー（複数可＝キック/スネア等）
    { name:"クラーベ", pitch:"E5", accentPitch:"E6",
      hits:[0, 1.5, 3, 5, 6],   // 打点＝拍位置（0.0起点。全体で beats*bars 拍ぶん）
      accents:[0] },            // hits のうちアクセント（accentPitch で鳴る）
  ],
}
```

- `hits`/`accents` は**拍単位の位置**（例 `1.5`＝2拍目の裏）。`grid` で割り切れる位置に置く。
- `hits*grid` が整数セル番号になり、`[0, beats*bars*grid)` の範囲に収まること。
- **音はドラム音源**（Tone.js の合成音。選択中の楽器とは独立）で鳴らします。層の `pitch` からドラムの種類を自動判定します：
  - `C2`/`C3` → **キック**、`A3`/`A4` → **スネア**、`F#5`/`F#6` → **ハイハット**、`E5`/`E6` → **クラーベ（木魚）**、`C5`/`C6` → **拍のクリック**。
  - `accents` に入れた打点は、そのドラムを強め（明るめ）に鳴らします。
  - 層に `drum:"kick"|"snare"|"hihat"|"clave"|"click"` を明示すると、`pitch` に関わらずその音で鳴らせます。
- 練習の `voicings`／`notes` のオフセットは **0〜19 がPCキーで弾ける範囲**（それ以上はマウス／ハイライトのみ）。
- 使えるコードタイプ（`type`）：`maj min dim aug sus4 sus2 maj7 min7 dom7 m7b5 six m6 mM7 add9`。これ以外（9th/11th/13th/オルタード等）は `keys`＋`voicing` で表現する。

# 準備編

## Part 1「さまざまな音名と音程」

1. **音名（幹音）** — ピアノの白鍵の音は、下から ド・レ・ミ・ファ・ソ・ラ・シ。 音楽理論では、これを英語で C・D・E・F・G・A・B と呼びます。　〔key C〕
2. **半音と全音** — となりの鍵盤（黒鍵もふくむ）までの距離が「半音」、半音2つ分が「全音」です。 半音は黒鍵をはさむとは限らず、ミ→ファ・シ→ドは白鍵どうしでも半音。　〔key C／楽譜 C-C♯-D-E-F にブラケット「半音／全音／半音」〕
3. **変化記号 ♯・♭・♮** — ♯は半音上げ、♭は半音下げ、♮はもとに戻す記号。 白鍵のあいだの黒鍵はこうしてできる音（C♯・E♭ は黒鍵、♮ で白鍵の E に戻す）。　〔key C／楽譜 C・C♯・E・E♭・E♮ にブラケット「♯半音上／♭半音下／♮戻す」〕
4. **異名同音** — 同じ高さの音でも、書き方によって名前が変わります。 たとえば C♯ と D♭ は同じ鍵盤の音。E♯＝F、B♯＝C なども同じ仲間です。　〔key C〕
5. **音程＝2音の距離（度数）** — 2つの音のへだたりを「度数」で表します。同じ音が1度、となりが2度…と数えます。 数えるときは両端の音もふくめます。例：C から E は C・D・E で「3度」。　〔key C〕
6. **完全・長短・増減** — 1・4・5・8度の音程は「完全」、2・3・6・7度は「長・短」と呼び分けます。 そこから半音広げると「増」、半音せまくすると「減」になります。　〔key C〕
7. **練習：ドレミファソラシド** — 下の鍵盤で、白鍵を左から順に弾いてドレミを鳴らしてみましょう。 「キー（左端の音）」を変えると、そのまま各調で練習できます。　〔key C／練習：ドレミファソラシド　notes:[0, 2, 4, 5, 7, 9, 11, 12]〕
8. **練習：音程（度数）を弾く** — 主音 C から、3度(E)・4度(F)・5度(G)・8度(C) と、へだたりを広げながら弾いてみましょう。　〔key C／練習：度数（3・4・5・8度）　notes:[0, 4, 5, 7, 12]〕

## Part 2「インターバルとスケール」

1. **スケールとは** — スケール（音階）は、主音から1オクターブ上まで、決まった順で音を並べたものです。 曲で使う音の土台、いわば「ものさし」になります。　〔key C〕
2. **メジャースケールの並び** — 明るい響きのメジャースケールは「全・全・半・全・全・全・半」の順で並びます。 半音になるのは「ミ→ファ」と「シ→ド」の2か所だけです。　〔key C〕
3. **Cメジャースケール** — Cメジャーは白鍵だけ（C D E F G A B C）で弾ける、いちばん基本のスケールです。　〔key C〕
4. **他キーは ♯/♭ が要る** — 同じ「全全半…」の並びを G から始めると、7番目の音に F♯ が必要になります。 こうして Gメジャー（G A B C D E F♯ G）ができます。　〔key G〕
5. **練習：メジャースケール（各キー）** — 白鍵を左から順に弾くとメジャースケールになります。 「キー」を変えて、いろいろな調で弾いてみましょう。　〔key C／練習：メジャースケール　notes:[0, 2, 4, 5, 7, 9, 11, 12]〕

## Part 3「さまざまなスケール」

1. **ナチュラルマイナー** — 暗い響きのマイナースケールは「全・半・全・全・半・全・全」の並びです。 Aマイナーは白鍵だけ（A B C D E F G A）で弾けます。　〔key A〕
2. **短調化（メジャー→マイナー）** — メジャースケールの 3・6・7番目の音を半音下げると、ナチュラルマイナーになります。 たった3つの音で、明るい響きが暗い響きに変わります。　〔key C〕
3. **平行調（relative）** — 使う音がまったく同じで、中心になる音（主音）だけが違う関係を「平行調」といいます。 Cメジャーと Aマイナーは同じ白鍵。中心が C か A かの違いだけです。　〔key C〕
4. **同主調（parallel）** — 主音が同じで、メジャーとマイナーだけが違う関係を「同主調」といいます。 Cメジャーと Cマイナー。他調から音を借りる（借用）ときの入り口です。　〔key C〕
5. **練習：Aマイナースケール** — マイナースケールの並びで弾いてみましょう（キーAなら左端が「ラ」）。　〔key A／練習：ナチュラルマイナースケール　notes:[0, 2, 3, 5, 7, 8, 10, 12]〕
6. **練習：同主調の明暗（C→Cm）** — 主音はそのまま、C（明るい）→ Cm（暗い）と弾き比べ。3度の音(E→E♭)だけで色が変わります。　〔key C／練習：同主調（C / Cm）　chords: C → Cm〕

## Part 4「五度圏と調号」

1. **調号とは** — 曲の間ずっと♯や♭が付く音を、毎回書かずに楽譜の左端へまとめたものが「調号」です。 たとえば Gメジャーは F にいつも♯が付くので、調号は♯1つになります。　〔key G〕
2. **♯が増える順番** — ♯は必ず F♯→C♯→G♯→D♯→A♯… の順で増えます（5度ずつ上がる並び）。 キーを5度上げるたびに♯が1つ増えます：C→G→D→A→E→B。　〔key C〕
3. **♭が増える順番** — ♭は B♭→E♭→A♭→D♭→G♭… の順（♯と逆＝4度ずつ上がる並び）。 キーを4度上げるたびに♭が1つ増えます：C→F→B♭→E♭→A♭。　〔key C〕
4. **主要キーの調号（早見）** — ♯系：G=♯1・D=♯2・A=♯3・E=♯4。 ♭系：F=♭1・B♭=♭2・E♭=♭3・A♭=♭4。 「一番右の♯の半音上」がメジャーの主音。♭系は「最後から2つ目の♭」が主音です。　〔key C〕
5. **平行調（同じ調号）** — 同じ調号を共有するメジャーとマイナーを「平行調」といいます（主音は短3度ちがい）。 調号なし＝C と Am。♯1つ＝G と Em。　〔key A〕
6. **練習：五度圏を回る** — C→G→D→A→E→B と、各キーの主音を弾いてみましょう（五度圏の右回り）。　〔key C／練習：五度圏の主音（C G D A E B）　notes:[0, 7, 2, 9, 4, 11]〕

## Part 5「倍音と音色」

1. **音の三要素** — 音は「大きさ（振幅）」「高さ（周波数）」「音色（波形）」の3つで決まります。 同じ高さ・大きさでも、波形が変われば楽器の違いとして聞こえます。　〔key C〕
2. **倍音列** — 1つの音の上には、整数倍の周波数の音（倍音）がふくまれています。 C を基音にすると、上に C・G・C・E・G・B♭・C… と倍音が積み上がります。　〔key C〕
3. **低い倍音はメジャーコード** — 下の方の倍音を拾うと、ちょうど C・E・G＝メジャーコードになります。 コードが自然で心地よく響くのは、この倍音の並びが理由のひとつです。　〔key C〕
4. **オクターブ＝周波数2倍** — ある音の1オクターブ上は、周波数がちょうど2倍。だから同じ音名（C→C）に聞こえます。　〔key C〕
5. **エンベロープ（ADSR）** — 音量の時間変化を4つで表すのが ADSR：Attack(立ち上がり)・Decay(減衰)・Sustain(持続)・Release(余韻)。 同じ波形でも ADSR が変わると、ピアノ的にも弦的にも聞こえます。　〔key C〕
6. **練習：倍音がつくる和音** — 低い倍音がつくる「ド・ミ・ソ」を弾いてみましょう。　〔key C／練習：倍音のド・ミ・ソ・ド　notes:[0, 4, 7, 12]〕


# メロディ編

## Part 6「メロディの基礎」

1. **モチーフ：提示・展開・解消** — メロディは短い動機（モチーフ）を「提示」し、変形して「展開」、反復を止めて「解消（終止）」します。 ひとつのアイデアをふくらませるのが、覚えやすいメロディのコツです。　〔key C〕
2. **順次進行と跳躍進行** — となりの音へ動くのが「順次進行」（なめらか）、離れた音へ跳ぶのが「跳躍進行」（劇的）。 上行は緊張、下行は安らぎ。とくに跳躍上行がいちばんドラマチックです。　〔key C〕
3. **跳躍（大きく跳ぶ）** — 同じ音でも、間を跳び越すと勢いが出ます。C から G、C へと跳んでみましょう。　〔key C〕
4. **音名と階名（相対で考える）** — 同じ「C」でも、Cメジャーでは主音（ド）、Fメジャーでは5番目の音（ソ）。 大切なのは絶対の高さより「キーの中心から見た位置」です。　〔key C〕
5. **空白（休符）の効果** — 音を止める「空白」も表現です。無音が静けさや語りかけを生みます（引き算の美学）。　〔key C〕
6. **練習：順次進行のメロディ** — となりの音へなめらかに動く順次進行のメロディを弾いてみましょう。　〔key C／練習：順次進行（ドレミファミレド）　notes:[0, 2, 4, 5, 4, 2, 0]〕
7. **練習：跳躍進行のメロディ** — 音を跳び越える跳躍進行。ド→ソ→ド→ミ と、勢いのある跳躍を弾いてみましょう。　〔key C／練習：跳躍進行（ド→ソ→ド→ミ）　notes:[0, 7, 12, 4, 7, 0]〕

## Part 7「傾性と調性引力」

1. **安定音と不安定音** — メジャースケールは、安定音 ド・ミ・ソ（C E G）と、不安定音 レ・ファ・ラ・シ（D F A B）に分かれます。 不安定音は、近くの安定音へ「進みたがる」性質（傾性）を持ちます。　〔key C〕
2. **導音の解決（シ→ド）** — 第7音 シ(B) は、半音上の主音 ド(C) へ強く引かれます。これが終止感の正体です。　〔key C〕
3. **ファ→ミ の解決** — 強い傾性を持つのは F と B。半音で隣の安定音へ解決します（F→E、B→C）。　〔key C〕
4. **収束と発散** — 中心音 C へ近づく動きが「収束」（弛緩）、遠ざかる動きが「発散」（緊張）。 G→A→B→C も F→E→D→C も、向きは逆でも同じ『Cへの収束』です。　〔key C〕
5. **カーネル（各音の個性）** — C＝完全な着地点、G＝次に安定、E＝情感、D＝浮遊、F＝最も不安定で下がりたい、B＝ドへ上がりたい。 各音の『キャラ』を知ると、狙って感情を作れます。　〔key C〕
6. **練習：傾性を解決する** — 不安定音を近くの安定音へ。シ→ド・ファ→ミ・レ→ド・ラ→ソ を順に弾きましょう。　〔key C／練習：傾性の解決（シ→ド・ファ→ミ・レ→ド・ラ→ソ）　notes:[11, 12, 5, 4, 2, 0, 9, 7]〕

## Part 8「ペンタトニックと民族音階」

1. **四七抜き＝メジャーペンタ** — メジャースケールから傾性の強い ファ(4)・シ(7) を抜いた5音が「メジャーペンタトニック」。 C D E G A。半音がなく、明るく外れにくい万能スケールです。　〔key C〕

## Part 13「シェル理論とハモリ」

1. **メロディの水平と垂直** — メロディは「時間の流れ（水平）」と「同時に鳴るコードとの関係（垂直）」の両面を持ちます。 コードトーンは澄み、非コードトーンは濁る——この濁りが表情になります。　〔key C〕
2. **シェル＝コードに対する度数** — メロディ音がコードの何度かで性質が決まります。Cコード上では C=1・E=3・G=5・B=7。 安定の序列は Root→5th→3rd→7th（力強い→豊か）。　〔key C〕
3. **偶数シェルは解決を求める** — 2・4・6度（D・F・A）はテンション。近くのコードトーンへ解決したがります。 例：D→C、F→E、A→G。　〔key C〕
4. **3度・6度のハモリ** — 主メロに3度上（または下）を重ねると、やわらかい彩りのハモリになります。 メロ C に上3度 E、下3度 A。3度と6度は転回関係の親戚です。　〔key C〕
5. **声部進行：反行・斜行・並行** — 2声の動きは3種類：逆向き（反行＝力強い）、片方だけ動く（斜行）、同じ向き（並行＝一体感）。 下の例は外側の2声が開いていく反行です。　〔key C〕
6. **練習：Cコードのシェル** — Cコードに対する R・3・5・7（ド・ミ・ソ・シ）を弾いて響きの違いを感じましょう。　〔key C／練習：シェル R-3-5-7　notes:[0, 4, 7, 11]〕

# コード編

## Part 14「ダイアトニックコード①」

1. **コード＝3度の積み重ね** — コード（和音）は、1つおきの音を3つ重ねて作ります（三和音）。 下から ルート（根音）・3度・5度。たとえば C は C・E・G です。　〔key C〕
2. **メジャーとマイナー** — 明るいメジャーは「長3度＋短3度」、暗いマイナーは「短3度＋長3度」。 変わるのは真ん中の3度の音だけ。ここで明暗が決まります。　〔key C〕
3. **ダイアトニックトライアド** — スケールの各音の上に三和音を積むと、そのキーで使える7つのコードになります。 Cメジャー：C・Dm・Em・F・G・Am・Bdim。　〔key C〕
4. **度数（ローマ数字）** — コードを番号（ローマ数字）で呼ぶと、キーが変わっても同じ理屈で考えられます。 大文字＝メジャー、小文字＝マイナー、°＝ディミニッシュ。　〔key C〕
5. **練習：ダイアトニック（トライアド）** — Cメジャーの7つのコードを、順番に弾いてみましょう。　〔key C／練習：ダイアトニックコード　chords: C → Dm → Em → F → G → Am → Bdim〕

## Part 15「ダイアトニックコード②」

1. **4和音＝7thを足す** — 三和音の上に、もう1つ3度を重ねると4和音（セブンスコード）になります。 たとえば Cmaj7 は C・E・G・B。厚みのある響きになります。　〔key C〕
2. **4種類の7th** — 同じ7thでも4種類。maj7＝メジャー＋長7度、m7＝マイナー＋短7度。 7（ドミナント7th）＝メジャー＋短7度、m7♭5＝ディミニッシュ＋短7度。　〔key C〕
3. **上手な数え方** — 見分けるコツは2つ。①土台の三和音がメジャーかマイナーか。 ②7度がルートの半音下（長7）か、全音下（短7）か。この2点でわかります。　〔key C〕
4. **ダイアトニック7th** — Cメジャーの7th：Imaj7・IIm7・IIIm7・IVmaj7・V7・VIm7・VIIm7♭5。 Vの位置だけがドミナント7th（G7）になります。　〔key C〕
5. **練習：ダイアトニック7th** — Cメジャーの7つのセブンスコードを順に弾いてみましょう。　〔key C／練習：ダイアトニック7th　chords: Cmaj7 → Dm7 → Em7 → Fmaj7 → G7 → Am7 → Bm7♭5〕

## Part 16「コード進行①」

1. **進行の力＝ルートの動き** — コードの進み方で最も強いのは、ルートが4度上がる（＝5度下がる）動きです。 たとえば C→F、G→C。次へ「進んだ」感じが強く出ます。　〔key C〕
2. **基本のスリーコード** — I・IV・V の3つは、コード進行の土台です。 I-IV-V-I が、いちばん基本の流れになります。　〔key C〕
3. **王道進行など** — I-V-vi-IV（C-G-Am-F）は、ポップスで大人気の「王道進行」。 ほかに I-vi-IV-V や ii-V-I もよく使われます。　〔key C〕
4. **カノン進行** — I-V-vi-iii-IV-I-IV-V は「カノン進行」。 ベースがなめらかに下がっていく、感動的な進行です。　〔key C〕
5. **練習：I-IV-V-I** — いちばん基本の I-IV-V-I を弾いてみましょう。　〔key C／練習：I-IV-V-I　chords: C → F → G → C〕
6. **練習：王道進行 I-V-vi-IV** — ポップスで大人気の C-G-Am-F。明るさと切なさが同居。　〔key C／練習：王道進行 I-V-vi-IV　chords: C → G → Am → F〕
7. **練習：カノン進行** — ベースがなめらかに下がる C-G-Am-Em-F-C-F-G。　〔key C／練習：カノン進行　chords: C → G → Am → Em → F → C → F → G〕

## Part 17「転回形の基本」

1. **転回形とは** — コードは、構成音を積む順番を変えても同じコードです。これを「転回形」といいます。 変わるのは、いちばん下の音（ベース）だけです。　〔key C〕
2. **三和音の3通り** — 三和音には3通りの積み方があります。基本形（ルートが最低音）。 第1転回形（3度が最低音）、第2転回形（5度が最低音）。　〔key C〕
3. **分数（スラッシュ）表記** — 転回形は「コード名／ベース音」の形で書きます。 Cの第1転回形は C/E、第2転回形は C/G です。　〔key C〕
4. **練習：Cの転回形** — 基本形→第1転回→第2転回の順に、光った鍵盤（最低音）に注意して弾こう。　〔key C／練習：Cの転回形　voicings: [[0,4,7],[4,7,12],[7,12,16]]〕

## Part 18「転回形の利用」

1. **目的はベースの滑らかさ** — 転回形を使う目的は、ベースの動きをなめらかにすること（ボイスリーディング）。 音の移動が少なくなり、上品で自然な進行になります。　〔key C〕
2. **下降ベースの例** — C→G/B→Am→C/G→F は、ベースが C-B-A-G… となめらかに下がります。 カノン進行などでよく使う、定番の形です。　〔key C〕
3. **上行ベースの例** — C→C/E→F のように、転回形でベースを上げていくこともできます。　〔key C〕
4. **練習：C → G/B → Am → C/G → F（下行）** — ベースが C-B-A-G-F と下がる動きを感じながら弾く。　〔key C／練習：転回形でつなぐ（下行）　voicings: [[0,4,7],[11,14,19],[9,12,16],[7,12,16],[5,9,12]]〕

## Part 19「コードの機能と代理コード①」

1. **3つの機能** — コードには3つの役割（機能）があります。トニック(T)＝安定、サブドミナント(SD)＝展開、ドミナント(D)＝緊張。 D→T と進むと「解決」した感じになります。　〔key C〕
2. **機能ごとのコード** — 同じ機能を持つコードがあります。トニック：I・IIIm・VIm。 サブドミナント：IV・IIm。 ドミナント：V・VIIdim。　〔key C〕
3. **ケーデンス（終止）** — フレーズの終わり方を「ケーデンス（終止）」といいます。 完全終止 V→I、変終止 IV→I、半終止 →V、偽終止 V→VIm。　〔key C〕
4. **練習：ii-V-I** — サブドミナント→ドミナント→トニックの ii-V-I を弾いてみましょう。　〔key C／練習：ii-V-I　chords: Dm → G → C〕
5. **練習：完全終止 V→I** — G→C。いちばん強く「終わった」と感じる終止。　〔key C／練習：完全終止 V→I　chords: G → C〕
6. **練習：変終止 IV→I** — F→C。やわらかい「アーメン終止」。　〔key C／練習：変終止 IV→I　chords: F → C〕
7. **練習：半終止 I→V** — C→G。ドミナントで止め「続きがある」宙づり感を残す。　〔key C／練習：半終止 →V　chords: C → G〕
8. **練習：偽終止 V→VIm** — G→Am。I へ解決すると思わせて VIm へ。意外性のある終止。　〔key C／練習：偽終止 V→VIm　chords: G → Am〕

## Part 20「コードの機能と代理コード②」

1. **代理コードとは** — あるコードを、同じ機能の別のコードに置きかえることを「代理」といいます。 共通する音が多いほど、自然に置きかえられます。　〔key C〕
2. **代表的な代理** — トニック同士 I⇄IIIm⇄VIm。サブドミナント同士 IV⇄IIm。 ドミナント同士 V⇄VIIdim。　〔key C〕
3. **注意点** — IIImは中間的な性格で、代理が弱くなる場面もあります。 V→VIm（偽終止）は、あえて解決を外して意外性を出す使い方です。　〔key C〕
4. **練習：I-IIm-V-VIm** — 代理コードを使った I-IIm-V-VIm を弾いてみましょう。　〔key C／練習：代理を使った進行　chords: C → Dm → G → Am〕
5. **練習：トニック代理 I-IIIm-VIm** — 同じ機能の C→Em→Am を弾き比べ。共通音が多くなめらか。　〔key C／練習：トニック代理（I-IIIm-VIm）　chords: C → Em → Am〕
6. **練習：サブドミナント代理 IV⇄IIm** — F を Dm に置きかえた F→Dm→G→C。　〔key C／練習：SD代理（IV⇄IIm）　chords: F → Dm → G → C〕

## Part 21「ドミナントモーション」

1. **V7→I が最強の解決** — V7→I は、もっとも強く「解決」を感じるコードの動きです。 ルートが4度上がり、さらに緊張した音（トライトーン）がほどけます。　〔key C〕

3. **なぜ強いか** — 「シ」と「ファ」は不安定な音程（トライトーン）です。 これが安定した音へ動くことが、あらゆる解決の核になっています。　〔key C〕
4. **練習：G7 → C** — いちばん強い解決 G7→C を弾いてみましょう。　〔key C／練習：ドミナントモーション　chords: G7 → C〕
5. **練習：トライトーンの解決** — G7 の核＝ファ＋シだけを弾き、次に ミ＋ド へ。2音が半音で解決する瞬間を体感。　〔key C／練習：トライトーン解決（ファ・シ→ミ・ド）　voicings: [[5,11],[4,12]]〕

## Part 22「マイナースケールのドミナント」

1. **自然短調の問題** — ナチュラルマイナーだと、Vのコードがマイナー(Vm)になります。 導音（主音の半音下の音）が無いため、ドミナントの緊張感が弱くなります。　〔key A〕
2. **ハーモニックマイナー** — そこで第7音を半音上げ、導音を作ります（Aマイナーなら G→G♯）。 すると V が V7（E7）になり、強い解決が使えるようになります。　〔key A〕
3. **メロディックマイナー** — ハーモニックマイナーは6→7度が広くて歌いにくいので、上行では6度も上げます。 これがメロディックマイナー。下行では元に戻すのが古典的です。　〔key A〕
4. **練習：Aハーモニックマイナー** — 第7音を上げたハーモニックマイナーの並びで弾いてみましょう。　〔key A／練習：ハーモニックマイナー　notes:[0, 2, 3, 5, 7, 8, 11, 12]〕
5. **練習：Aメロディックマイナー（上行）** — 第6音も上げた上行形（A B C D E F♯ G♯ A）。増2度が消えなめらか。　〔key A／練習：メロディックマイナー（上行）　notes:[0, 2, 3, 5, 7, 9, 11, 12]〕
6. **練習：E7 → Am（マイナーの解決）** — 導音 G♯ を持つ E7 から Am へ。マイナーでも強い解決が使える。　〔key C／練習：V7→Im（E7→Am）　chords: E7 → Am〕

## Part 23「セカンダリドミナント」

1. **考え方** — キーの中のあるコードを、一時的に「そのキーの I」とみなします。 そのIへ向かう V7 を借りてくるのが「セカンダリドミナント」（V7/x）です。　〔key C〕
2. **作れる対象（5つすべて）** — dim 以外の各ダイアトニックコードへ V7 を作れます（V7/II〜V7/VI の5つ）。 A7→Dm・B7→Em・C7→F・D7→G・E7→Am。　〔key C／楽譜：V7/II〜VI の5対象すべてを V7→対象で表示〕
3. **使い方** — 向かいたいコードの直前に置くと、そのコードが強調され、色づきます。　〔key C〕
4. **練習：C → E7 → Am（V7/VI）** — C→E7→Am。E7 が次の Am を引き立てます。　〔key C／練習：セカンダリドミナント V7/VI　chords: C → E7 → Am〕
5. **練習：C → D7 → G（V7/V）** — D7（＝V7/V）が、ドミナントの G を強調。　〔key C／練習：セカンダリドミナント V7/V　chords: C → D7 → G〕
6. **練習：C → A7 → Dm（V7/II）** — A7（＝V7/II）でマイナーの Dm を引き立てる。　〔key C／練習：セカンダリドミナント V7/II　chords: C → A7 → Dm〕

## Part 24「ツーファイブワン」

1. **ii-V-I の基本** — サブドミナント(ii)→ドミナント(V)→トニック(I) の流れが ii-V-I。 Dm7-G7-Cmaj7。ジャズで最もよく使う進行です。　〔key C〕
2. **リレイテッドⅡm7** — セカンダリドミナントの前に、その完全5度上のマイナー7thを足せます。 例：E7 の前に Bm7 を置くと、Bm7-E7-Am というミニ ii-V になります。　〔key C〕
3. **マイナーなら IIm7♭5** — 向かう先がマイナーコードのときは、IIを m7♭5 にします。 例：Am へ向かうなら Bm7♭5-E7-Am。　〔key C〕
4. **練習：Dm7-G7-Cmaj7（メジャー）** — ジャズの基本 Dm7-G7-Cmaj7 を弾いてみましょう。　〔key C／練習：ii-V-I（メジャー）　chords: Dm7 → G7 → Cmaj7〕
5. **練習：Bm7♭5-E7-Am（マイナー）** — 向かう先がマイナーなら II を m7♭5 に。マイナー ii-V-i。　〔key C／練習：ii-V-i（マイナー）　chords: Bm7♭5 → E7 → Am〕
6. **練習：リレイテッドⅡ（Bm7-E7-Am）** — E7 の前に Bm7 を足したミニ ii-V。　〔key C／練習：リレイテッドⅡm7　chords: Bm7 → E7 → Am〕

## Part 25「コード進行②」

1. **実例①** — C - A7 - Dm - G7 - C。 A7（＝V7/II）が、次の Dm を引き立てています。　〔key C〕
2. **実例②（リレイテッドⅡ）** — C - Em7 - A7 - Dm7 - G7 - C。 Em7-A7 のミニ ii-V が、Dm7 へなめらかにつなぎます。　〔key C〕
3. **ツーファイブの連続** — ii-V をくり返しながら下がっていく、ジャズの定番パターンです。　〔key C〕
4. **練習：C-A7-Dm7-G7-C（実例①）** — セカンダリドミナント A7 を使った進行。　〔key C／練習：実例①（V7/II）　chords: C → A7 → Dm7 → G7 → C〕
5. **練習：C-Em7-A7-Dm7-G7-C（実例②）** — Em7-A7 のミニ ii-V を挟んだ実例。　〔key C／練習：実例②（リレイテッドⅡ）　chords: C → Em7 → A7 → Dm7 → G7 → C〕

## Part 26「パッシングディミニッシュ」

1. **パッシングディミニッシュとは** — 全音はなれた2つのコードの間に、半音上のディミニッシュを挟むテクニックです。 ベースが半音でつながり、なめらかな流れになります。　〔key C〕
2. **上行型（定番）** — I-♯Idim-IIm、IV-♯IVdim-V などが定番。 例：C-C♯dim-Dm7。　〔key C〕
3. **仕組み** — dim にふくまれる緊張音が、次のコードの音へ解決します（C♯→D、G→F）。 下がっていく形もありますが、上がる形がよく使われます。　〔key C〕
4. **練習：C → C♯dim → Dm7（I-♯Idim-IIm）** — 半音でつなぐ上行型。　〔key C／練習：パッシングディミニッシュ（I-♯Idim-IIm）　chords: C → C#dim → Dm7〕
5. **練習：F → F♯dim → G（IV-♯IVdim-V）** — もう一つの定番。ベースが半音で上がり V へ。　〔key C／練習：パッシングディミニッシュ（IV-♯IVdim-V）　chords: F → F#dim → G〕

## Part 27「五度圏の利用」

1. **五度圏とは** — 調やコードを、完全5度ずつ円に並べたのが「五度圏」です。 時計回りに進むと♯が1つずつ増え、反時計回りに進むと♭が増えます。　〔key C〕
2. **調号の整理** — C(♯♭なし)→G(♯1)→D(♯2)→A(♯3)…と、5度ごとに調号が1つずつ増えます。 Gメジャーは♯が1つ（F♯）だけの調です。　〔key G〕
3. **ii-V-I も五度圏で動く** — ii-V-I は、ルートが反時計回り（4度上がる方向）に進みます。 五度圏を見ると、ツーファイブの連続がひと目でわかります。　〔key C〕
4. **練習：五度圏を右回り（トニック巡り）** — 時計回りに C→G→D→A→E とトニックを巡る。　〔key C／練習：五度圏（右回り）　chords: C → G → D → A → E〕
5. **練習：五度圏を左回り（ii-V-I）** — ルートが4度上がる Dm7-G7-Cmaj7 が五度圏を左へ進む。　〔key C／練習：五度圏（左回り＝ii-V-I）　chords: Dm7 → G7 → Cmaj7〕

## Part 28「裏コード①」

1. **裏コード＝トライトーン代理** — V7 を、共通のトライトーンを持つ「♭II7」に置きかえるのが「裏コード」です。 G7→C なら、G7 の代わりに D♭7 を使えます。　〔key C〕
2. **共有トライトーン** — G7 の「シ・ファ」と D♭7 の「ファ・ド♭(=シ)」は、まったく同じ2音です。 この共通音があるから、置きかえても解決感が保たれます。　〔key C〕
3. **効果** — ベースが D♭→C と半音で下がり、おしゃれでなめらかな響きになります。　〔key C〕
4. **練習：Dm7 → D♭7 → C** — 半音で下がる裏コードを弾いてみましょう（D♭7 は C♯7 と同じ音）。　〔key C／練習：裏コード　chords: Dm7 → C#7 → C〕
5. **練習：表と裏を弾き比べ** — 同じ C へ、G7（表）→C と D♭7（裏）→C。ベースの動きの違いを聴き比べる。　〔key C／練習：表(G7)と裏(D♭7)　chords: G7 → C → C#7 → C〕

## Part 29「サブドミナント終止とサブドミナントマイナー」

1. **サブドミナント終止（変終止）** — IV→I で終わる終止を「変終止（プラガル）」といいます。 「アーメン終止」とも呼ばれ、やわらかく落ち着く響きです。　〔key C〕
2. **サブドミナントマイナー(SDM)** — 同主短調（Cマイナー）から借りるサブドミナントが「サブドミナントマイナー」。 代表は IVm。IV→IVm→I とすると、切ない終止感が出ます。　〔key C〕
3. **SDMの仲間** — ほかに ♭VI・♭VII・IIm♭5 などがあります。 ♭VI→♭VII→I は、余韻の残るサビ終わりの定番です。　〔key C〕
4. **練習：F → C（変終止）** — 基本の IV→I。やわらかい「アーメン終止」。　〔key C／練習：変終止 IV→I　chords: F → C〕
5. **練習：F → Fm → C（SDM）** — 同主短調から借りた IVm の切ない響き。　〔key C／練習：サブドミナントマイナー　chords: F → Fm → C〕
6. **練習：♭VI → ♭VII → I** — A♭→B♭→C。余韻の残るサビ終わりの定番進行。　〔key C／練習：♭VI-♭VII-I　chords: G#(=A♭) → A#(=B♭) → C〕

## Part 30「クリシェとペダルポイント」

1. **クリシェ** — コードはほぼ保ったまま、内側の音を半音ずつ動かすラインが「クリシェ」です。 例：Cm-CmM7-Cm7-Cm6（中で C-B-B♭-A と下がる線が動く）。　〔key C〕
2. **ペダルポイント** — ベース（や特定の音）を鳴らし続けたまま、上のコードだけを動かす手法です。 主音を保つトニックペダル、属音を保つドミナントペダルが定番です。　〔key C〕
3. **効果** — 動きの中に「変わらない軸」ができ、統一感や緊張の持続が生まれます。 （上の楽譜は、クリシェの動く線 C-B-B♭-A）　〔key C〕
4. **練習：Cm クリシェ（マイナー）** — 内声が C-B-B♭-A と半音で下がる Cm のクリシェ。　〔key C／練習：クリシェ（マイナー）　chords: Cm → CmM7 → Cm7 → Cm6〕
5. **練習：C クリシェ（メジャー）** — メジャーでも同じ手法。C→Cmaj7→C7→C6（上の線が B→B♭→A）。　〔key C／練習：クリシェ（メジャー）　chords: C → Cmaj7 → C7 → C6〕

## Part 31「sus4系分数コード」

1. **sus4 の復習** — sus4 は、3度の代わりに完全4度を使うコード。宙づりのような響きです。 Gsus4→G のように、3度へ動いて解決したくなります。　〔key C〕
2. **sus4系分数コード** — ベース音の上に別の和音を重ねて、sus4 のような響きを作る分数コードです。 例：Dm7/G、F/G、C/D。　〔key C〕
3. **使い方** — V の手前や V の代わりに置くと、やわらかいドミナントの響きになります。　〔key C〕
4. **練習：Gsus4 → C** — sus4 の宙づり感から、C へ解決する流れを弾いてみましょう。　〔key C／練習：sus4分数（近似）　chords: Gsus4 → C〕
5. **練習：F/G → C（sus4系分数）** — G の上に F の三和音を重ねた F/G（G・F・A・C）→ C へ解決。　〔key G／練習：F/G → C　voicings: [[0,10,14,17],[5,9,12]]〕

## Part 32「add9と9thコード」

1. **テンションとは** — コードの土台の音(1,3,5,7)の上に重ねる、色づけの音が「テンション」です。 オクターブ＋2・4・6度＝9th・11th・13th。　〔key C〕
2. **add9** — 三和音に9thだけを足したコード（7thは入れません）。 Cadd9＝C・E・G・D。明るく広がる響きです。　〔key C〕
3. **9thコード** — セブンスコードに、さらに9thを重ねたのが9thコードです。 Cmaj9・C9・Cm9 など。おしゃれで厚みのある響きになります。　〔key C〕
4. **練習：Cadd9** — Cadd9（C・E・G・D）の、きらびやかな響きを弾く（7thは入れません）。　〔key C／練習：add9　chords: Cadd9〕
5. **練習：Cmaj9** — Cmaj7 に9thを足した Cmaj9（C・E・G・B・D）。厚みのある響き。　〔key C／練習：Cmaj9　voicings: [[0,4,7,11,14]]〕
6. **練習：C9（ドミナント9th）** — C7 に9thを足した C9（C・E・G・B♭・D）。ファンキーな響き。　〔key C／練習：C9　voicings: [[0,4,7,10,14]]〕

## Part 33「11thコードと13thコード」

1. **11thと13th** — テンションはさらに上に積めます。11th＝4度上、13th＝6度上の音です。　〔key C〕
2. **アヴォイドノート** — コードの音と半音でぶつかり、濁って聞こえる音を「アヴォイドノート」といいます。 メジャー系の11thは3度とぶつかるため、そのままでは使いません。　〔key C〕
3. **対処** — 11th は ♯11 に変えるか、sus4 にして3度を抜くと使えます。 13th は上のほうに置くと、きれいに響きます。　〔key C〕
4. **練習：C7sus4（3度を抜いて回避）** — 対処1。3度を抜いた sus4 で11度（完全4度）を活かす。　〔key C／練習：sus4で11thを活かす　chords: Csus4〕
5. **練習：Cmaj7(♯11)（♯11で回避）** — 対処2。11thを半音上げた♯11(F♯) なら3度とぶつからない。　〔key C／練習：Cmaj7(♯11)　voicings: [[0,4,7,11,18]]〕

## Part 34「様々なテンションのテクニック」

1. **アヴェイラブルテンション** — コード（の機能）ごとに、使えるテンションと避けるテンションが決まっています。 例：Imaj7 では 9th・13th が使え、11th は避けます。　〔key C〕
2. **回避テク** — アヴォイドを避けるコツ：3度を抜く(sus)、♯11・♯9 に変える、上のほうに置く。　〔key C〕
3. **コード別の目安** — Imaj7：9,13。IIm7：9,11,13。 V7：9,13＋オルタード。IVmaj7：9,♯11,13。　〔key C〕
4. **練習：Cmaj9（9thの響き）** — C E G B D を弾いて、9th の広い響きを確かめましょう。　〔key C／練習：maj9　voicings: [[0,4,7,11,14]]〕
5. **練習：C6/9（9と13）** — Imaj7 で使える 9th(D)・13th(A) を同時に足した C6/9（C E G A D）。　〔key C／練習：C6/9（9,13）　voicings: [[0,4,7,9,14]]〕

## Part 35「オルタードテンションの基本」

1. **オルタードテンションとは** — ドミナント7thにつく、変化したテンションを「オルタードテンション」といいます。 ♭9・♯9・♯11・♭13 の4つです。　〔key C〕
2. **位置の確認** — ルートから見て ♭9(半音上)・♯9・♯11(＝減5度)・♭13(＝増5度) の位置にあります。　〔key C〕
3. **使いどころ** — 解決するドミナント（とくにマイナーへ向かう V7）で、緊張をぐっと強めます。　〔key C〕
4. **G7(♭9) の響き** — G B D F A♭（♭9＝A♭）。鍵盤と楽譜のアニメで確認しましょう。　〔key C〕
5. **練習：G7(♭9)** — G B D F A♭ を弾いて、♭9 の張りつめた緊張感を確かめる（マイナーへ向かう V7 で活躍）。　〔key G／練習：G7(♭9)　voicings: [[0,4,7,10,13]]〕

## Part 36「オルタードテンションの性質」

1. **オルタードスケール** — オルタードスケールは、メロディックマイナーの第7モードです。 ドミナントのオルタードテンションが、全部そろうスケールです。　〔key C〕
2. **テンションリゾルブ** — テンションは、解決先のコードの音へなめらかに進みます。 例：♭9(A♭) は次の主音 G へ、半音で下がります。　〔key C〕
3. **♭13と♯5** — ♭13 は増5度（♯5）と同じ音。オーギュメント系の響きとつながります。　〔key C〕
4. **G7(♯9) の響き** — G B D F A♯（♯9＝A♯）。ロックでおなじみの、力強い緊張感です。　〔key C〕
5. **練習：G7(♯9)** — G B D F A♯ を弾く。長3度と♯9が同居する、ジミヘン・コードの荒々しい響き。　〔key G／練習：G7(♯9)　voicings: [[0,4,7,10,15]]〕
6. **練習：オルタードスケール** — オルタードが全部そろうスケール（G A♭ B♭ B D♭ E♭ F）。V7 上で使う。　〔key G／練習：オルタードスケール　notes:[0, 1, 3, 4, 6, 8, 10, 12]〕

## Part 37「テンションの複合」

1. **複合の基本** — 相性のよいテンションを、組み合わせて重ねます（9th と 13th など）。　〔key C〕
2. **注意点** — アヴォイド同士や、半音でぶつかる音は避けます。 詰め込みすぎず、音の間隔をあけて配置するのがコツです。　〔key C〕
3. **代表的な組** — V7(♭9,♭13)、V7(9,13)、maj7(9,13) などがよく使われます。　〔key C〕
4. **G7(9,13) の響き** — G B F A E（9＝A, 13＝E）。おだやかで洗練された響きです。　〔key C〕
5. **練習：G9 → G7(9,13)** — まず G9（G B D F A）、次に5度を抜き13を足した G7(9,13)（G B F A E）。テンションを重ねる感覚を掴む。　〔key G／練習：テンションの複合（G9→G13）　voicings: [[0,4,7,10,14],[0,4,10,14,21]]（最上音13＝E はマウス可）〕

## Part 38「ポリコード、アッパーストラクチャートライアド」

1. **ポリコード** — 2つのコードを上下に重ねて表す書き方を「ポリコード」といいます。 例：D / C（下が C、上が D）。　〔key C〕
2. **アッパーストラクチャートライアド(UST)** — セブンスコードの上に、テンションでできる三和音を乗せる形です。 例：D/C7＝C7(9,♯11)。上の三和音が、一気に華やかなテンションを与えます。　〔key C〕
3. **定番のUST（ドミナント上）** — ドミナント上では II（9,♯11）、♭VI（♭13,♭9）、♭II（♭9,♭5）などが定番です。　〔key C〕
4. **D/C7 の響き** — 下が C7、上が Dメジャー。広がりのある、豊かなテンションの響きです。　〔key C〕
5. **練習：D/C7（UST）** — C7 の土台（C E B♭）に D の三和音（D F♯）を乗せた C7(9,♯11)。　〔key C／練習：D/C7＝C7(9,♯11)　voicings: [[0,4,10,14,18]]〕

## Part 39「更に特殊なテンション」

1. **10th・12th・14thとは** — オクターブ＋3度＝10th、＋5度＝12th、＋7度＝14th。 これらはコードの音の重複なので、ふつうはコードネームに書きません。　〔key C〕
2. **使う場面** — 特別なボイシングや、響きを細かく指定したいときに、まれに登場します。　〔key C〕
3. **要点** — 覚えるべき標準テンションは 9・11・13 の3つです。 10・12・14 は「コードの音を1オクターブ上げたもの」と知っておけば十分です。　〔key C〕
4. **練習：標準テンション 9・11・13** — 土台の C から 9th(D)・11th(F)・13th(A) を順に弾いて位置を確かめる。　〔key C／練習：9th・11th・13th（D・F・A）　notes:[0, 2, 5, 9]〕

## Part 40「その他の特殊なコード」

1. **aug の復習** — オーギュメント(aug)は、長3度を2つ重ねたコードです。 ここでは、クリシェ以外の使い方を見ていきます。　〔key C〕
2. **ドミナント代理としての aug** — V7 の5度を半音上げた Vaug は、強いドミナントの響きになります。 Gaug→C のように、解決に使えます。　〔key C〕
3. **分数aug（ブラックアダー／イキスギ）** — aug をドミナント代理にして、ルートを増4度下に置いた分数コードです。 例：Gaug/D♭（次が C）。ベースが半音で動き、劇的な響きになります。　〔key C〕
4. **練習：Gaug → C** — Gaug から C への、劇的な解決を弾いてみましょう。　〔key C／練習：aug の解決　chords: Gaug → C〕

## Part 41「ドミナントの本質（トライトーンと5度下降）」

1. **ドミナントモーションの2要素** — ドミナントを使いこなす鍵は「ドミナントモーション」。①トライトーンの解消、②ルートの完全5度下降の2要素が推進力。　〔key C／楽譜 V7→I／応用例 V7→I〕
2. **トライトーン＝鍵盤6個分** — ドミナント7thの「3rd」と「♭7th」の間隔は鍵盤ちょうど6個分（増4度＝減5度）。この宙づりの落差が原動力。　〔key C／楽譜 G7 と 3rd・♭7th〕
3. **トライトーンの解消** — G7 の シ(B)・ファ(F) が次の C の ド(C)・ミ(E) へ半音で解決。外へ／内へ開く2音の動きが終止感の核。　〔key C〕
4. **ルートの完全5度下降** — もう一つの要素は、ルートが5度下（＝4度上）へ動くこと（G→C）。「コードが変わった」という実感の源。　〔key C〕
5. **練習：G7 → C（2要素を体感）** — トライトーンの解消とルート5度下降の両方を感じて弾く。　〔key C／練習：ドミナントモーション　chords: G7 → C〕
6. **練習：トライトーンの解消だけ** — ファ＋シだけを弾き、次に ミ＋ド へ。2音の半音解決を取り出す。　〔key C／練習：ファ・シ→ミ・ド　voicings: [[5,11],[4,12]]〕

## Part 42「いろいろなドミナント」

1. **プライマリードミナント** — キーの中に最初から在るドミナント7th＝V7（Cメジャーなら G7）。ダイアトニック唯一のドミナント7th。　〔key C〕
2. **セカンダリードミナント（復習）** — キー外からドミナントを借り、各ダイアトニックへドミナントモーション（例 D7＝V7/V）。詳細は Part 23。　〔key C〕
3. **ドッペルドミナント（ダブルドミナント）** — 進行先もドミナント7thの状態。濁りが連続する。D7→G7→C（V7/V の解決先 G を G7 化）。　〔key C／応用例 D7→G7→C〕
4. **エクステンデッドドミナント** — 終着点から逆算し、ドミナントモーションを3〜4連続で繋ぐ。E7→A7→D7→G7→C。　〔key C／応用例 E7→A7→D7→G7→C〕
5. **テンション入りドミナント** — ナチュラル系(9,13)やオルタード系(♭9,♯9,♯11,♭13)で濁らせ、解決のコントラストを強める（詳細 Part 35〜37）。　〔key C〕
6. **練習：ドッペルドミナント** — D7→G7→C を弾く。　〔key C／練習：ドッペルドミナント　chords: D7 → G7 → C〕
7. **練習：エクステンデッドドミナント** — E7→A7→D7→G7→C を弾く。　〔key C／練習：エクステンデッドドミナント　chords: E7 → A7 → D7 → G7 → C〕

## Part 43「偽終止とピカルディ終止」

1. **偽終止（デセプティブ・ケイデンス）** — 期待の I を裏切り別のコードへ。V7 からトニック代理（VIm・IIIm）へ（G7→Am、G7→Em）。　〔key C／応用例 V7→VIm〕
2. **セカンダリードミナントの偽終止** — セカンダリの解決先を代理コードへすり替える。C7（V7/IV）は F へ行くはずを IIm（Dm）へ。　〔key C〕
3. **ピカルディ終止** — マイナー曲の最後を強引にメジャーに。Am…→A（メジャー）。最後だけ晴れやかに解決。　〔key A／応用例 Am…→A〕
4. **練習：偽終止 G7→Am** — I へ行くと思わせて VIm へ。　〔key C／練習：偽終止 V7→VIm　chords: G7 → Am〕
5. **練習：セカンダリの偽終止 C7→Dm** — C7（V7/IV）を本来の F ではなく代理 Dm へ。　〔key C／練習：セカンダリの偽終止　chords: C7 → Dm〕
6. **練習：ピカルディ終止 E7→A** — マイナーの流れから最後だけメジャーの A へ。　〔key C／練習：ピカルディ終止　chords: E7 → A〕

## Part 44「特殊な構造（7sus4・ハイブリッド）」

1. **サスペンデッド・ドミナント（7sus4）** — 3rdの代わりに4thを使いトライトーンを含まない。定義からは外れるが、文脈でドミナント代用（G7sus4→C）。　〔key C〕
2. **7sus4 の解決** — 4th(C)を3rd(B)へ動かせば G7 に戻り C へ。動かさず sus のまま進んでもOK。　〔key C〕
3. **ハイブリッドコード（フュージョンコード）** — ルートの全音下のメジャーを上に乗せる（IV/V 等）。G の全音下＝F で F/G。リッチで現代的。　〔key C〕
4. **練習：G7sus4 → C** — 3度を持たない宙づりから C へ解決。　〔key C／練習：サスペンデッド・ドミナント　chords: Gsus4 → C〕
5. **練習：F/G → C（ハイブリッド）** — G の上に F 三和音を重ねた F/G→C。　〔key G／練習：ハイブリッド F/G → C　voicings: [[0,10,14,17],[5,9,12]]〕

## Part 45「スペシャルファンクション・ドミナント」

1. **スペシャルファンクションとは** — ドミナントモーション（5度下降）に従わない特殊なドミナント7th。代表6つ：I7・IV7・♭VII7・♭VI7・II7・VII7。　〔key C〕
2. **I7（トニックの7th化）** — I をセブンス化した C7。ブルースの「ホット」で土くさい雰囲気。　〔key C／応用例 I7-IV7-I7（ブルース）〕
3. **IV7（サブドミナントの7th化）** — IV をセブンス化した F7。ブルースで I7 と並んで多用。　〔key C〕
4. **♭VII7 → I（直進ドミナント）** — ♭VII7（B♭7）はトニックへ直進できる有名で使いやすいコード。ミクソリディアン的な解決。　〔key C〕
5. **♭VI7 → I** — ♭VI7（A♭7）もトニックへ直進可。裏コードにも通じる半音の色気。　〔key C〕
6. **II7（同じ形でも役割ちがい）** — II7（D7）はドッペルドミナントと同形だが、5度下降せず I 等へ進む使い方を指す。　〔key C〕
7. **VII7** — VII7（B7）は IV7 の代理的役割で、トニックへ進める。　〔key C〕
8. **練習：♭VII7 → I** — トニックへ直進する定番 B♭7→C。　〔key C／練習：♭VII7→I　chords: A#(=B♭)7 → C〕
9. **練習：I7 → IV7（ブルース）** — トニックもサブドミナントも7th化。C7→F7→C7。　〔key C／練習：I7-IV7-I7　chords: C7 → F7 → C7〕
10. **練習：♭VI7 → I** — もう一つの直進ドミナント A♭7→C。　〔key C／練習：♭VI7→I　chords: G#(=A♭)7 → C〕

## Part 46「高度な応用テクニック」

1. **整列式2連続ドミナント** — ドミナントの枠を、同じ解決先の2コード（表と裏）に分割して並べる。G7→D♭7→C。　〔key C／応用例 G7→D♭7→C〕
2. **オルタレーション** — 構成音（主にテンション）を意図的に変化させ意外性を突く。G7 を G7(♭9,♯9) に。　〔key C／楽譜 G7(♭9,♯9)〕
3. **非機能的・適当な配置** — ルールをあえて無視してドミナントを置き、調性をぼかす。例：C7→E♭7→F♯7→A7 と短3度で平行移動。　〔key C／応用例 平行ドミナント〕
4. **練習：整列式2連続 G7→D♭7→C** — 同じ C へ解決する表と裏を並べる。　〔key C／練習：整列式2連続ドミナント　chords: G7 → C#7(=D♭7) → C〕
5. **練習：非機能パラレルドミナント** — 短3度ずつ平行移動する C7→E♭7→F♯7→A7。　〔key C／練習：非機能パラレルドミナント　chords: C7 → D#7 → F#7 → A7〕


# リズム編

## Part 47「拍と拍子」

1. **拍（ビート）とは** — 音楽の時間を刻む等間隔のパルスが「拍（ビート）」。手拍子や足踏みのタイミングです。 下のグリッドが1拍ずつ光り、1拍目にアクセントが付きます。　〔key C／リズム：♩=96／4拍×1小節／grid 4／拍[0 1 2 3]〕
2. **4/4拍子** — 拍を4つずつまとめたのが「4/4拍子」。ポップスの大多数がこれです。 強・弱・中強・弱（1拍目が強拍、3拍目が中強拍）という重みの階層があります。　〔key C／リズム：♩=100／4拍×1小節／grid 4／4/4[0 1 2 3]〕
3. **3/4拍子（ワルツ）** — 3拍で1周する拍子。「強・弱・弱」でくり返す、ワルツの優雅な揺れです。　〔key C／リズム：♩=132／3拍×1小節／grid 4／3/4[0 1 2]〕
4. **2/4拍子（マーチ）** — 2拍で1周する拍子。「強・弱」でくり返す、行進やサンバのノリです。　〔key C／リズム：♩=108／2拍×1小節／grid 4／2/4[0 1]〕
5. **キックとスネアの基本** — ドラムの土台は、1・3拍目のキック（低）と、2・4拍目のスネア（高）。 この組み合わせが、ロックやポップスの基本ビートです。　〔key C／リズム：♩=100／4拍×1小節／grid 4／キック[0 2]／スネア[1 3]〕

## Part 48「ビートとサブディビジョン」

1. **サブディビジョン（拍の分割）** — 1拍をさらに等分することを「サブディビジョン」といいます。 1拍を2分割＝8分（8ビート）、4分割＝16分（16ビート）。細かいほど疾走感が出ます。　〔key C／リズム：♩=96／4拍×1小節／grid 4／拍[0 1 2 3]〕
2. **8ビート** — 1拍を2分割し、8分音符でハイハットを刻むのが8ビート。ロック／ポップスの定番です。　〔key C／リズム：♩=100／4拍×1小節／grid 4／ハイハット[0 0.5 1 1.5 2 2.5 3 3.5]／キック[0 2]／スネア[1 3]〕
3. **16ビート** — 1拍を4分割して刻むのが16ビート。ファンクやR&Bの細かくうねるノリです。　〔key C／リズム：♩=92／4拍×1小節／grid 4／16分[0 0.25 0.5 0.75 1 1.25 1.5 1.75 2 2.25 2.5 2.75 3 3.25 3.5 3.75]〕
4. **表拍と裏拍** — 1拍の前半が「表拍（ダウンビート）」、後半が「裏拍（アップビート）」。 裏拍を強調すると軽快で浮遊感のあるノリ（テクノ／ディスコ）になります。　〔key C／リズム：♩=104／4拍×1小節／grid 4／表拍[0 1 2 3]／裏拍[0.5 1.5 2.5 3.5]〕
5. **四つ打ち＋裏ハイハット** — 全拍でキックを打つ「四つ打ち」に、裏拍のハイハットを重ねるとダンス系のノリに。　〔key C／リズム：♩=124／4拍×1小節／grid 4／キック[0 1 2 3]／ハイハット[0.5 1.5 2.5 3.5]〕

## Part 49「強拍とシンコペーション」

1. **強拍・弱拍・中強拍** — 4/4では1拍目＝強拍、3拍目＝中強拍、2・4拍目＝弱拍という重みの階層があります。　〔key C／リズム：♩=96／4拍×1小節／grid 4／重み[0 1 2 3]〕
2. **バックビート** — 本来弱い2・4拍目にスネアで強いアクセントを置くのが「バックビート」。 ロック／ソウルの推進力の源です。　〔key C／リズム：♩=100／4拍×1小節／grid 4／キック[0 2]／スネア[1 3]〕
3. **シンコペーション** — 拍の重みがない位置（裏拍など）にアクセントを置く技法。 前のめりの緊張が生まれ、強拍へ戻ると解決します。下は表拍とシンコペの比較です。　〔key C／リズム：♩=96／4拍×1小節／grid 4／オンビート[0 1 2 3]／シンコペ[0 1.5 2.5]〕
4. **「移民の歌」風** — 4拍目の16分裏にキックを突っ込む、前のめりな有名パターン（ダドンッ）。　〔key C／リズム：♩=108／4拍×1小節／grid 4／キック[0 1 2 3 3.75]／スネア[1 3]〕

## Part 50「クラーヴェ」

1. **トレシーロ（3-3-2）** — 1小節の8分音符を 3+3+2 でグループ化したラテンの基本リズム。 打点は 0・1.5・3拍目。ラテン／レゲトン／ポップスに広く使われます。　〔key C／リズム：♩=100／4拍×1小節／grid 2／3-3-2[0 1.5 3]〕
2. **ソン・クラーベ（3-2）** — 2小節で1周するラテンの核。1小節目に3打（トレシーロ側）、2小節目に2打。 ラテン音楽全体の『ものさし』となるリズムです。　〔key C／リズム：♩=100／4拍×2小節／grid 2／クラーベ[0 1.5 3 5 6]〕
3. **ソン・クラーベ（2-3）** — 3-2 を前後反転したもの。1小節目の頭が空くので、大人っぽく落ち着いた出だしに。　〔key C／リズム：♩=100／4拍×2小節／grid 2／クラーベ[1 2 4 5.5 7]〕
4. **ボサ・クラーベ** — ソン・クラーベの2打側の後打を『3拍目の裏』へずらし、シンコペを強めたもの。 ボサノヴァのしなやかな揺れが生まれます。　〔key C／リズム：♩=108／4拍×2小節／grid 2／ボサ[0 1.5 3 5 6.5]〕

## Part 51「シャッフルと複合拍子」

1. **三連符** — 1拍を3等分したのが三連符。ジャズやブルースの『跳ねる』土台になります。　〔key C／リズム：♩=92／4拍×1小節／grid 3／三連[0 0.333 0.667 1 1.333 1.667 2 2.333 2.667 3 3.333 3.667]〕
2. **シャッフル** — 三連符の1音目と3音目だけを鳴らす（2:1の比）と、『タッカ タッカ』と跳ねます。 低いテンポほど跳ねが強く、速いテンポでは等分割に近づきます。　〔key C／リズム：♩=92／4拍×1小節／grid 3／シャッフル[0 0.667 1 1.667 2 2.667 3 3.667]／スネア[1 3]〕
3. **6/8拍子（複合拍子）** — 8分音符6つを『3＋3』にまとめた拍子。大きく2拍・中で3分割の二重構造です。 1つ目と4つ目の8分にアクセント（付点4分が2つ）。　〔key C／リズム：♩=120／6拍×1小節／grid 2／6/8[0 1 2 3 4 5]〕
4. **ハーフタイム・シャッフル** — シャッフルのままスネアを半分に減らし（3拍目のみ）、どっしり遅く感じさせる技法。　〔key C／リズム：♩=92／4拍×1小節／grid 3／シャッフル[0 0.667 1 1.667 2 2.667 3 3.667]／スネア[2]〕

## Part 52「ポリリズムと変拍子」

1. **ポリリズム 3:2** — 同じ長さの中で、一方が3等分・もう一方が2等分を同時に鳴らすのが「ポリリズム」。 3:2 は最も基本的で、ヘミオラとも呼ばれます。　〔key C／リズム：♩=90／2拍×1小節／grid 3／2の層[0 1]／3の層[0 0.667 1.333]〕
2. **ポリリズム 4:5** — 4の層と5の層を重ねると、20分割で1周する複雑なうねり（Perfume『ポリリズム』風）。　〔key C／リズム：♩=100／4拍×1小節／grid 5／4の層[0 1 2 3]／5の層[0 0.8 1.6 2.4 3.2]〕
3. **変拍子 5/4（3+2）** — 5拍を『3＋2』にグループ分けした変拍子。総拍数よりアクセントの位置が大切です。　〔key C／リズム：♩=120／5拍×1小節／grid 2／5/4[0 1 2 3 4]〕
4. **変拍子 7/8（2+2+3）** — 8分音符7つを『2＋2＋3』に分けた変拍子。頭を強調しフレーズを反復して馴染ませます。　〔key C／リズム：♩=150／7拍×1小節／grid 2／7/8[0 1 2 3 4 5 6]〕
